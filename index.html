How This Website Was Made

Built with simple HTML, CSS, and GitHub Pages

This site is a clean, minimal example of what you can do with just a single index.html file and free hosting on GitHub Pages.

We wrote the HTML and CSS by hand, uploaded it to a GitHub repository, then enabled GitHub Pages to publish it online for free.

If you want to create your own site like this, here’s a sample code snippet you can start with:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>My Sample Site</title>
</head>
<body>
  <h1>Hello, World!</h1>
  <p>This is my first website.</p>
</body>
</html>

Just save that as index.html, upload it to your GitHub repo, and enable GitHub Pages — easy as that!
